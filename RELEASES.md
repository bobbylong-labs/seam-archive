# Release Index  

This file tracks immutable releases within the seam-archive repository.  

Rules:  
- Each release corresponds to a timestamped snapshot.  
- Releases cannot be edited once published.  
- Each release must be accompanied by a verification hash.  
- Transcript additions are batched into numbered releases.  
- DOI assignment (optional) attaches only to completed releases.

- Release 001 — Initialization Shell (PROVENANCE.md set)  
- Release 002 — Transcript Intake (GPT-4)
